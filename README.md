# STed2 clone

現在開発中のため、リリース版は存在しません。

## 概要
本プロジェクトは、TURBO 氏（故 戸田貴之氏）によって Sharp X680x0 シリーズ向けに開発された MIDI シーケンサ STed (STed2) のクローンアプリケーションです。<br />
原版の画面や動作を再現しながら、Windows / macOS / Linux / iOS / Android の各環境で動作するクロスプラットフォームアプリケーションとして再実装することを目的としています。

## 謝辞
原版を開発された TURBO 氏、STed の公開場所となった草の根 BBS「ゆいNET」の SysOp である、ゆい工房（山本裕一郎）氏、原版ソースコードの公開に関わられた BOMBER 氏、その他、STed に関わられた全ての方々に深く感謝いたします。
